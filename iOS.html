课表2.0 ⎡L!NG⎦

我的课表改名为L!NG了.

这三个多月没更新都干了什么

人呐就都不知道，自己就不可以预料。一个人的命运啊，当然要靠自我奋斗，但是也要考虑到历史的行程。我绝对不知道，我好端端的代码开启iCloud文稿后怎么就不见了，所以之后找了好久,直到放寒假后才找到，自言自语“我已经决定了，还是得用GitHub”，但是钱怎么办呢。我实在我也不是谦虚，我怎么有钱用GitHub咧？但是呢，GitHub有个“学生大礼包”，所以后来我就念了两句诗，叫“Shut up，take my money”，那么我就继续写代码开始开发了。
寒假我这几十天也没干什么别的，大概就是完成之前定的三个需求

  • 本地化,资瓷英文和中文
  • 滋瓷自定义JavaScript 代码以提供安全的一键导入功能
  • 定制化表盘挂件

如果说还有一点什么改进那就是增加了文档中心。因为中间一段时间在折腾网站和CMS，所以实际开发时间也就一个多月。还有Bug修复也是很重要的。但这些都是次要的，我主要的我就是这三个改进，很惭愧，就做了一点微小的改进。
我知道这里洋文好的人多得很,所以也希望大家能给英文翻译提个意见.
目前2.0.1版本已经发布，2.1版本正在审核。

下载地址
L!NG - My Extensible School Timetable
https://itunes.apple.com/cn/app/l-ng-my-extensible-school-timetable/id1164016936?l=en&mt=8

名称由来

在开发新版本的⎡我的课表⎦的URL Scheme时,原定的URL是mytimetable://,然而在Safari试验时发现已经有叫MyTimetable的网站了,而且功能很相似,不过人家是Web,为了日后不和别人冲突(说不定哪天别人就做iOS了呢),就决定给App换个名字.

⎡L!NG⎦这个名字最初是由School Bell这个词想到的,School Bell -> 铃 -> ling -> L!NG,这个名称应该没人用,不过缺点就是不太好搜.

iOS打造自定义输入法

最近需要输入音标,一时间没有找到合适的App,于是就考虑自己写一个简单的音标输入法

要求:Swift 2,Xcode 7

  • Let's Code

概述

APP Extension

App Extension允许用户在整个系统(iOS/OSX)使用App的一些功能,其中就包括自定义键盘.

自定义键盘

自定义键盘在苹果官方文档里已经介绍得比较详细了.
主要有以下几点注意:

  • 如果只是为你自己的App而打造的输入键盘,那就不要用自定义键盘,iOS提供了更好的选择: Custom Views for Data Input in Text Programming Guide for iOS.
  • 自定义键盘必须要能够切换到别的键盘
  • 自定义键盘在有时候是不能输入的(例如输入安全信息时,电话键盘...)
  • 自定义键盘是无法像系统键盘一样超出键盘范围范围
  • 一大堆限制,加密,麦克风等都不行
